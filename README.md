Exocortic - Distribution packaging for the Zuri project
=======================================================

Copyright (c) 2024-2026, Michael Frank

Overview
--------

Exocortic contains the build system and supporting scripts to construct a
redistributable package containing all of the executables, libraries, data and
configuration to build and run Zuri programs.

Prerequisites
-------------

Exocortic requires the following dependencies to be installed on the system:
```
CMake version 3.27 or greater:      https://cmake.org
Conan version 2:                    https://conan.io
Python version 3:                   https://www.python.org
Tempo:                              https://github.com/msfrank/tempo
Lyric:                              https://github.com/msfrank/lyric
Zuri:                               https://github.com/msfrank/zuri
```
